# Echoes of Bathala

This project is a Massively Multiplayer Online Role Playing Game (MMORPG) with Blockchain integration that has Filipino Mythology as the basis for its lore. The following are the Major Components separated by purpose:

**Game Architecture**

1. Game Client
2. Game Server
3. Database

**Token Infrastructure**

4. Smart Contract
5. Blockchain API Gateway
6. Crypto Request Router
9. Wallet Service

**Maintenance**

7. Asset storage
8. Analytics dashboard

To start development clone this repo.

```bash
git clone https://github.com/Markism-JA/Echoes-of-Bathala.git
```
