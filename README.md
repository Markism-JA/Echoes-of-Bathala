# Echoes of Bathala

This project is a Massively Multiplayer Online Role Playing Game (MMORPG) with Blockchain integration that has Filipino Mythology as the basis for its lore. The following are the Major Components separated by purpose:

## Overview

**Game Architecture**

1. Game Client
2. Game Server
3. Database

**Token Infrastructure**

1. Smart Contract
2. Blockchain API Gateway
3. Crypto Request Router
4. Wallet Service

**Maintenance (To be created depending on how the project changes)**

1. Asset storage
2. Analytics dashboard
